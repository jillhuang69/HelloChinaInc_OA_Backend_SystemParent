## Project Overview

This repository contains the backend implementation of the **Internal Office Automation (OA) System** developed for Hello China Inc.

The system is designed as an internal enterprise application to support office administration, approval workflows, role-based access control (RBAC), and integration with a workflow engine. The backend exposes RESTful APIs consumed by the administrative frontend and employee-facing interfaces.

This project is intended for internal use and is not exposed to the public internet.

---

## Technology Stack

- **Backend Framework:** Java Spring Boot  
- **Persistence:** MyBatis / MyBatis Plus  
- **Security:** Spring Security, JWT-based authentication  
- **Workflow Engine:** Activiti  
- **Database:** MySQL  
- **Cache:** Redis  
- **Build Tool:** Maven
